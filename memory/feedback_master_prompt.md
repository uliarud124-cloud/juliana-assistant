## СТИЛЬ — 12 ПРАВИЛ (из разбора 100 постов, группа «Джулиа точка роста»)

1. КОРОТКИЕ ПРЕДЛОЖЕНИЯ. Иногда одна фраза на строке. Создаёт ритм.
2. ПРОВОКАЦИОННЫЙ ЗАГОЛОВОК: преувеличение + ирония + неожиданный поворот
3. ПУНКТУАЦИЯ: многоточие... пауза. Тире — вместо сложных конструкций. Скобки (вот так) — мысль в сторону. «)» вместо смайлика.
4. САМОИРОНИЯ: «счастливчик моё второе имя», «пишет пост в последнюю минуту» 😅
5. КОНКРЕТИКА: не «один клиент» — а «таролог», «нутрициолог». Не «отдыхала» — а «уронила телефон в Крыму».
6. СТРУКТУРА экспертного поста: крючок → разворот → список → вывод → мягкий CTA
7. ЭМОДЗИ: 💙 или 🔹 как маркер списка, 😅 для самоиронии. Никаких 🎉🔥💥
8. ЛАЙФ — без фильтра: «трясущиеся руки», «злая и растерянная» — настоящесть
9. CTA: «пиши РАЗБОР» / «пиши БОТ» — слово-триггер. Иногда просто вопрос.
10. ТЕРМИНЫ объяснять сразу: «трипваер (недорогой продукт)», «лид-магнит — подарок»
11. ТЕМП неравномерный — это нормально. Возвращаться с объяснением.
12. ПОЗИЦИЯ своя: когда все паникуют — она спокойно разбирается.

ЗАПРЕЩЕНО: «приобрести по ссылке», «успейте», «ограниченное предложение», длинные абзацы без разбивки, слово «контент» без иронии, слабые переходы «и вот тут»/«а потом», слова «баг»/«умно».

НЕ злоупотреблять тире подряд (ИИ-стиль) — но само по себе тире в её стиле есть.

Структура поста с пользой (Рыжанок): заголовок → проблема узнаваемая → переход → решение пошагово → личный пример → CTA с реакцией.
